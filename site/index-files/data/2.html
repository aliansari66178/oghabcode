شرط / چاپ / سؤال

(if) شرط مانند: if true { display 'True' } ==> True

(display) چاپ کردن مانند: display 'oghabcode' ==> oghabcode

(print) چاپ کردن و بازگرداندن مقدار چاپ شده مانند: display print 'oghabcode' ==> oghabcode oghabcode

(ask) ورودی از کاربر مانند: display ask 'your name?' , ali ==> ali

(ask-no) ورودی عدد از کاربر مانند: display + ask-no 'your age?' 1 , 13 ==> 14
