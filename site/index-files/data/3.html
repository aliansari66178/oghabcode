متغیّر / آرایه

(set) تغییر متغیّر مانند: set 1 'oghab'

(get) خواندن متغیّر مانند: display get 1 ==> oghab

(arr) تعریف آرایه به تعداد خانه‌ی خاص مانند: set 1 arr 3 1 2 3

(+item) افزودن داده‌ به تعدادی خاص به آرایه مانند: set 1 +item get 1 2 4 5

(index) خواندن یک خانه‌ی آرایه که از ۱ شروع می‌شود مانند: display index get 1 4 ==> 4
