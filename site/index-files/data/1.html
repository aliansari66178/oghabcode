عملگر‌ها

(+\add) برای جمع دو عدد مانند: + ۲ ۳.۴ = ۵.۴

(+str\addstr) برای جمع دو رشته مانند: +str "oghab" "code" = oghabcode

(*\mul) برای ضرب دو عدد مانند: * ۱.۵ ۴ = ۶

(-\sub) برای تفریق دو عدد مانند: - ۸.۹ ۰.۰۱ = ۸.۸۹

(/\div) برای تقسیم دو عدد مانند: / ۷.۵ ۲.۵ = ۳

(!\fact) برای فاکتوریل یک عدد مانند: ! ۴ = ۲۴

(--\neg) برای منفی کردن یک عدد مانند: -- ۴ = -۴

(%\mod) برای باقیمانده دو عدد مانند: % ۱۱ ۳ = ۲

(^\pow) برای توان دو عدد مانند: ^ ۲ ۳ = ۸
